# 🏐 Set Vôlei

Aplicativo mobile desenvolvido com React Native (Expo) com o objetivo de facilitar o dia a dia para os professores e alunos da Escola de Vôlei Set Vôlei.

## 📱 Sobre o Projeto

O Set Vôlei é uma aplicação mobile cuja proposta é facilitar a integração entre professores e alunos e que centraliza informações sobre as atividades da escola, tais como informativos, mapa com informações de suas unidades, perfis de usuários (professor, admin e usuário), tabela de horário dos treinos e informações sobre planos de pagamento. 

---

## 🚀 Funcionalidades

- 📍 Visualização de quadras em mapa (geolocalização)
- 👤 Tela de perfil do usuário
- 🧭 Navegação por abas (tabs)
- 📱 Interface mobile responsiva
- 📌 Integração com GPS do dispositivo

---

## 🛠️ Tecnologias Utilizadas

- React Native
- Expo
- Expo Router
- React Native Maps
- Expo Location
- JavaScript
